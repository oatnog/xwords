Crossword Puzzle Generator
Created by Allan Anderson. (c) 2012
Inspiration and code borrowed from Brian Helmig.
Stylesheets borrowed from htmlboilerplate.com.

This is a simple web application which creates a non-deterministic crossword puzzle given a set of words and clues.
